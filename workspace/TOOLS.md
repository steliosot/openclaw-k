# TOOLS.md — Local Notes

Skills define *how* tools work (see `/app/skills/*/SKILL.md`).
This file is for *your* per-session notes — anything you want to
remember in the background that isn't appropriate for the shared
skill docs.

## What Goes Here

Jot anything you want to carry between turns:

- Reference files the user has uploaded and where you staged them
- Prompt phrasings that worked particularly well this session
- Project-specific style preferences the user has expressed
- Running brand identity notes (palette, voice, imagery do's/don'ts)
- Anything you'd otherwise have to re-derive from the transcript

## Why Separate?

Skills are shared across every Maestro user. This file is yours
alone. Keep them apart so skill updates don't overwrite your notes,
and so session-specific context doesn't bleed into the shared skill
docs.
